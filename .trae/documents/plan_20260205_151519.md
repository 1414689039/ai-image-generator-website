我将为您在灵感社区中添加“我的分享”功能，并支持删除自己的分享。

### 后端修改 (Worker)
1.  **修改获取列表接口 (`GET /api/gallery/list`)**：
    *   增加 `filter` 参数支持。
    *   当 `filter=my` 时，增加 SQL 查询条件 `AND user_id = current_user_id`，只返回当前用户发布的画作。
2.  **新增删除接口 (`DELETE /api/gallery/:id`)**：
    *   用于取消分享。
    *   验证当前用户是否为画作作者。
    *   将画作的 `is_public` 状态更新为 `0`（不物理删除，只是从社区下架）。

### 前端修改 (Frontend)
1.  **状态管理**：
    *   在 `Gallery.tsx` 中增加 `filter` 状态 (`'all' | 'my'`)，默认为 `'all'`。
2.  **UI 交互**：
    *   在页面顶部标题旁或筛选栏增加切换标签：[发现灵感] / [我的分享]。
    *   当切换到“我的分享”时，重新加载列表。
3.  **删除功能**：
    *   引入 `Trash2` 图标。
    *   在画廊卡片和详情弹窗中，如果当前画作属于自己 (`is_owner`)，显示删除/下架按钮。
    *   点击删除时弹出确认框，确认后调用后端接口并刷新列表。

### 执行步骤
1.  修改 `worker/src/routes/gallery.ts` 实现后端逻辑。
2.  部署后端 Worker。
3.  修改 `frontend/src/pages/Gallery.tsx` 实现前端 UI 和交互。
4.  构建并部署前端 Pages。
